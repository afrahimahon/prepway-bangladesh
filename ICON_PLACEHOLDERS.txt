HOW TO CONVERT SVG → PNG:
  1. Open https://svgtopng.com (free)
  2. Paste the SVG code from ICONS_REFERENCE.html
  3. Set size to 64x64 pixels
  4. Download and rename to the correct icon-XX.png filename
  5. Drop the file in this folder (same folder as index.html)

OR — use icons from a free icon set:
  https://phosphoricons.com  (clean, modern, free)
  https://lucide.dev         (minimal, developer-friendly)
  https://heroicons.com      (simple outlined)
  Download as SVG → convert to PNG → rename to icon-XX.png
