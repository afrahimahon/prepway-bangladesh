STEP 2 — GET YOUR ICONS
───────────────────────────────────
Option A — Use the built-in SVG codes (quickest):
  1. Open ICONS_REFERENCE.html in your browser
  2. You will see all 14 icons rendered visually
  3. Go to https://svgtopng.com
  4. Copy each <svg>...</svg> block → paste → download as PNG → rename

Option B — Download free icons (cleaner results):
  Recommended: https://phosphoricons.com
  - Search for the icon type (mic, play, book, etc.)
  - Download as SVG → open in browser → right-click → Save As PNG
  - Rename to icon-XX.png

For social icons (Facebook, Instagram, YouTube, Telegram, Messenger):
  Download official brand assets:
  - Facebook:  https://about.meta.com/brand/resources/facebook/logo/
  - Instagram: https://about.meta.com/brand/resources/instagram/icons/
  - YouTube:   https://www.youtube.com/howyoutubeworks/resources/brand-resources/
  - Telegram:  https://telegram.org/tour/screenshots
  - Messenger: https://about.meta.com/brand/resources/messenger/logo/

STEP 4 — DEPLOY TO VERCEL
───────────────────────────────────
  1. Go to https://vercel.com — sign up with Google (free)
  2. Click "Add New" → "Project"
  3. Click "Deploy from your computer" (or drag and drop)
  4. Drag this entire "prepway-bangladesh-site" FOLDER onto the page
  5. Vercel detects it as a static site — no settings needed
  6. Click "Deploy"
  7. Your site is live in ~30 seconds at:
        https://prepway-bangladesh-site.vercel.app

  To update later: just drag the folder again. Vercel redeploys automatically.


CUSTOM DOMAIN (LATER)
───────────────────────────────────
  When you buy a domain (e.g. prepwaybangladesh.com):
  1. Vercel project → Settings → Domains → Add your domain
  2. Vercel gives you 2 DNS records to paste into your domain registrar
  3. HTTPS is automatic and free

  Recommended registrars: Namecheap (~$10/year), Porkbun (~$8/year)


EDITING THE WEBSITE
───────────────────────────────────
  Best free editor: VS Code → https://code.visualstudio.com/
  - Open index.html in VS Code
  - Ctrl+F  →  search for any text
  - Ctrl+H  →  find and replace
  - Ctrl+S  →  save
  - Then re-upload the folder to Vercel
