Prepway Bangladesh — Icon Reference

All 14 icons are already included as SVG files in the icons/ folder. Replace any file to use your own icon.

How This Works

UI icons (icon-02 to icon-10) are black SVGs. CSS automatically flips them white in dark mode and keeps them dark in light mode. You only need one file per icon — it works for both themes.

Brand icons (icon-01, icon-11 to icon-14) are full-color and stay the same in both themes.

To replace an icon: find the file in the icons/ folder and replace it with your own file. Keep the same filename (e.g. icon-02.svg).

You can use .svg files directly (already done) or convert to PNG at svgtopng.com and rename to the same filename. SVG is recommended.

Want to use different icons?

Download free icons from phosphoricons.com or lucide.dev

1. Download as SVG  →  2. Rename to the correct icon-XX.svg filename  →  3. Drop into the icons/ folder  →  Done.

For UI icons (02-10): download the black/dark version. CSS handles the rest.

For brand icons (01, 11-14): use the full-color version.
